## Star-Travels
> Go语言编写的社区帖子web项目


## 框架选型
* Gin来实现路由
* gorm来操作mysql
* go-redis来操作redis
* viper来管理配置文件
* zap来记录日志
* validator来校验错误
* snowflake来生成ID
* jwt实现认证
* jaeger实现链路追踪

## 部署
* docker 部署项目
* docker 部署mysql一主三从
* docker 部署redis三主三从，sentinel集群 
* docker 部署jaeger
* [redis脚本](https://www.zhugeqing.top/docker/docker-redis/)
* [mysql脚本](https://www.zhugeqing.top/docker/docker-mysql/)
